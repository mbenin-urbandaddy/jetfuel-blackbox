<private> <static> blackbox.web.core.Controller.dispatch()
Dispatches a route pattern to a blackbox.web.view.Base class.

If currentView is unset (undefined), then we assume that this is the first dispatch event, indicating our initial page load. In this case, we DO NOT call the blackbox.web.view.Base#load() method - the only thing to do is to instantiate the view class.

If, however, currentView has been set, then we can assume this is a subsequent page request, so we DO call blackbox.web.view.Base#load() to request whichever resources we need to render the Page.
